# Новости Steam + Welcome с ролями + База стримеров

## 1. Новости Steam (автопостинг)

### Файл: `bot/rp-bot.js`

### Как работает:
- Каждые 30 минут парсит страницу SCUM в Steam: `https://store.steampowered.com/app/518150/SCUM/`
- Ищет свежие новости (апдейты, скидки, события)
- Сравнивает с кешем последних постов
- Если новость новая — постит в канал новостей (CHANNEL_NEWS) или в LOLKA

### Функция:
```js
async function checkSteamNews() {
  // Парсим RSS/HTML Steam
  // Сравниваем с кешем
  // Если новое — отправляем в канал
}
```

### Кеш:
- `data/steam-news.json` — хранит ID последних обработанных новостей
- Не дублирует уже опубликованное

## 2. Welcome с ролями (rp-bot.js)

### Когда новый участник заходит на сервер:
1. Основной бот (index.js) уже пишет приветствие
2. AI-бот пишет ОТДЕЛЬНО через 5-10 секунд:
```
👋 Привет! Я помощник сервера AMURKA PVE.

Скажи, ты здесь как:
🎮 Игрок — я помогу с подключением, модами, вопросами
📺 Стример — я добавлю тебя в нашу базу стримеров

Просто напиши "игрок" или "стример"!
```

### Логика обработки ответа:
```
Если ответ содержит "игрок" →
  - Выдать роль ROLE_PLAYERS
  - Ответить: "Готово! Ты теперь игрок AMURKA PVE. Пиши мне если нужна помощь!"

Если ответ содержит "стример" →
  - Выдать роль ROLE_STREAMERS
  - Спросить: "Крутно! Скинь ссылку на твой канал (VK Видео Live, YouTube, Twitch)"
  - Сохранить в базу стримеров
  - Добавить на сайт (обновить data/streamers.json)
```

### Env переменные:
- `ROLE_STREAMERS` — ID роли стримеров (добавить в .env)

## Верификация
1. Новый участник заходит → через 5 сек AI-бот пишет опрос
2. Участник пишет "игрок" → получает роль, бот отвечает
3. Участник пишет "стример" → получает роль, бот спрашивает ссылку
4. Ссылка получена → сохраняется в data/streamers.json
5. Каждые 30 минут бот проверяет Steam → новые новости постятся
6. Стримеры отображаются на сайте
