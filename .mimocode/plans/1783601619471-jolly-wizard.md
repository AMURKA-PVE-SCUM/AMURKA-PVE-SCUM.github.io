# План улучшений RP-бота

## Приоритетные улучшения

### 1. Контекст диалога
**Зачем:** Бот помнит что спрашивали ранее в разговоре, отвечает релевантно.

**Реализация:**
- Сохранять последние 5 сообщений пользователя в userHistory
- Добавлять историю в промпт для DeepSeek
- Пример: "А где торговец?" → "Какой? Оружия или еды?"

**Файлы:** `rp-bot.js` — функция getUserHistory, отправка в messages

### 2. Автообучение
**Зачем:** Когда вопрос задают 3+ раза — автоматически добавлять в базу знаний.

**Реализация:**
- Считать повторяющиеся вопросы в `data/question_counts.json`
- При достижении порога (3 раза) — добавлять в `data/knowledge.json`
- DeepSeek сам генерирует лучший ответ для базы
- При следующем таком вопросе — отвечать из базы без AI

**Файлы:** `rp-bot.js` — функции trackQuestion, checkAutoLearn

## Файлы для изменения
- `rp-bot.js` — основные изменения
- `data/question_counts.json` — счётчики вопросов (создаётся автоматически)

## Верификация
1. Запустить бота
2. Задать один вопрос 3 раза
3. Проверить что он добавился в knowledge.json
4. Задать вопрос снова — должен ответить из базы
5. Спросить "а где торговец?" после "покажи торговцев" — должен понять контекст
